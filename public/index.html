Meet the Hunters!

Simple app using React.js along with Redux.js 7.1, having full interaction with all of the components state by changing it based on what the user does on the interface. We used two reducers, combined into one, along with four action types and an instance of the dispatch to launch them when the user clicks on any of these buttons. Redux is cool, folks!
